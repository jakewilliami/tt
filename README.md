<h1 align="center">Tea Timer</h1>

## Description

A simple tea timer CLI written in Rust.  Will count up in seconds.

## Quick Start

```shell
$ just
$ ./tt -h
```

## Usage

```shell
$ tt -h
Tea timer!  Count up in seconds

Usage: tt

Options:
  -h, --help     Print help
  -V, --version  Print version
```
