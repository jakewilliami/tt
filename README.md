<h1 align="center">Tea Timer (`tt`)</h1>

## Description

A simple tea timer CLI written in Rust.  Will count up in seconds.

## Quick Start

```commandline
$ ./build.sh
$ ./tt -h
```

## Usage

```commandline
$ tt -h
Tea timer!  Count up in seconds

Usage: tt

Options:
  -h, --help     Print help
  -V, --version  Print version
```
